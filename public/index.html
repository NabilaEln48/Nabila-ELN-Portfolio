Flappy Bird Game

A 2D arcade-style game built in Unity using C#. Inspired by the original Flappy Bird, this version features real-time physics, animated sprites, dynamic obstacle spawning, and score tracking. This project helped strengthen my understanding of Unity scripting, game loop logic, and C# fundamentals for game development.
